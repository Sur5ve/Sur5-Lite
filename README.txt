BETA VERSION - PORTABLE AI ASSISTANT
============================

Welcome to Beta Version, your advanced AI desktop companion!
This USB drive contains everything needed to run powerful AI models on any Windows computer.

✨ NO INSTALLATION • NO INTERNET • NO API KEYS • NO ADMIN RIGHTS ✨

QUICK START:
1. Double-click: BetaVersion.exe
2. Wait for AI model to load (first time only - about 1-2 minutes)
3. Start intelligent conversations with your local AI!

WHAT'S INCLUDED:
- BetaVersion.exe (Main AI application - ~45MB)
- gemma-3-1b-it-Q4_K_M.gguf (AI brain - ~800MB)
- README.txt (This guide)

ESSENTIAL HOTKEYS:
- Ctrl + S: Send your message to AI
- Ctrl + Z: Stop AI response generation
- Ctrl + F: Search through conversation history
- Ctrl + X: Clear all conversations
- Ctrl + P: Modify AI personality/instructions
- Ctrl + Mouse Wheel: Zoom interface

AMAZING FEATURES:
✓ Source Word Highlighting - Words from your prompts are highlighted in AI responses
✓ Conversation Save/Load - Preserve complete AI sessions (File menu)
✓ Model Switching - Use different AI brains (File > Select Model...)
✓ Document Upload - Drag PDFs, Word docs for AI analysis
✓ Completely Offline - Zero internet dependency
✓ Ultra Portable - Runs from USB on any Windows PC

SYSTEM REQUIREMENTS:
- Windows 7 or newer (any version)
- 2GB RAM minimum (4GB+ for best performance)
- Any CPU type (Intel, AMD, ARM - no GPU needed)
- 1GB+ free space for AI models

PERFORMANCE EXPECTATIONS:
- Gemma 3-1B: ~20 responses per second on modern CPUs
- First launch: Model loads into memory (patience required!)
- After loading: Near-instant AI responses
- Larger models = smarter responses but slower speed

TROUBLESHOOTING:
❓ Slow responses? Close other programs to free up RAM
❓ App crashed? Restart BetaVersion.exe - it will reload automatically
❓ Want different AI? File > Select Model to choose new GGUF files
❓ Need help? Check conversation history for tips and examples

DOCUMENT INTELLIGENCE:
Drag and drop these file types for AI analysis:
• PDF documents
• Word files (.docx)
• Text files (.txt)
• Markdown (.md)
• HTML pages

The AI can summarize, analyze, answer questions, and extract information from your documents!

ADVANCED TIPS:
• Use specific, detailed prompts for better AI responses
• Save important conversations before closing (File > Save Chat)
• Try different system prompts to change AI personality
• Experiment with various GGUF models for different capabilities
• Use Ctrl+F to find information in long conversations
